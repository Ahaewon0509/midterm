1. 파이어베이스 서비스의 종류 및 특징

- 특징: 손쉽게 백엔드 구축이 가능하다<br>
구글에서 관리하므로 여러ㅏ지 서비스 구글 서비스 연동이 쉽다.<br>
따로 관리 필요없이 하나오 통제가능하다.<br>
- 서비스 종류
인증(Authentication): 사용자 로그인 및 ID 관리<br>
클라우드 함수(Cloud Functions): 서버 없이 이벤트 위주로 동작하는 백엔드<br>
클라우드 파이어스토어(Cloud Firestore): 클라우드에 호스팅 된 실시간의 비관계형<br>(NoSQL) 데이터베이스<br>
클라우드 스토리지(Cloud Storage): 거대하게 확장할 수 있는 파일 스토리지<br>
파이어베이스 호스팅(Firebase Hosting): 전 세계를 대상으로 한 웹 호스팅<br>
머신러닝 키트(ML Kit): 일반적인 머신러닝(ML) 작업을 위한 SDK<br>
실시간 데이터베이스(Realtime Database): 클라우드에 호스팅 된 실시간의 비관계형<br>(NoSQL) 데이터베이스<br>
